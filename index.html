Windows Shortcuts

  • Ctrl + CCopy line
  • Alt + ↑ / ↓Move line
  • Shift + Alt + ↑ / ↓Duplicate line
  • Ctrl + Shift + KDelete line
  • Ctrl + EnterInsert line below
  • Ctrl + DSelect next word
  • Ctrl + Shift + LSelect all matches
  • Ctrl + SpaceSuggestions
  • Shift + Alt + FFormat file
  • Ctrl + PQuick open
  • Ctrl + GGo to line
  • Ctrl + `Toggle terminal
  • Ctrl + \\Split editor
